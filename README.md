This is a Games Library app.

### Technologies / Libraries ###
Backend:
- Python 3.9.6
- Flask 2.0.1
- Flask-Cors 3.0.10

Frontend:
- Vue 3.2.13
- Bootstrap 5.3.3
- Bootswatch 4.5.2 Sketchy Theme

### Run the app ###

# backend

## Change directory
```
cd ./backend
```

## Run the server
```
pipenv run python main.py
```

# frontend

## Change directory
```
cd ./frontend
```

## Project setup
```
npm install
```

## Compiles and hot-reloads for development
```
npm run serve
```

## Compiles and minifies for production
```
npm run build
```

## Lints and fixes files
```
npm run lint
```
